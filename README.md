# GitHub Pages Academic Website Starter

This is a simple static GitHub Pages starter site for an academic profile.

## Deployment steps

1. Create a GitHub repository named exactly:

   `<your-github-username>.github.io`

2. Upload all files from this folder into the repository root.

3. Go to repository `Settings` > `Pages`.

4. Under `Build and deployment`, select:

   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. Save. The site should publish at:

   `https://<your-github-username>.github.io`

## Files

- `index.html`: main website content.
- `assets/style.css`: website styling.
- `cv.pdf`: optional CV file to add later.

## Before publishing

Replace all placeholder links and publication entries with exact information.
Do not publish private information such as phone number, home address, immigration details, funding documents, or unpublished confidential project material.
